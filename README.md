## Sponsor the Rich

This project investigates how private vs. government-owned cultural venues influence the presence of public art in Vancouver. Surprisingly, private ownership was **positively associated** with higher public art density—especially in high-density neighborhoods—challenging the belief that public art is solely government-driven.

Coded in **R using Jupyter Notebook**, the analysis included socio-economic controls and robustness checks.

The title *Sponsor the Rich* reflects the finding that **privately owned venues—often seen as profit-driven—actually enrich the public urban environment** by fostering cultural vibrancy.
